# 💬 Análise de Sentimentos com Azure AI

Este projeto faz parte do desafio prático da DIO e mostra como utilizar o **Language Studio**, da Azure, para identificar sentimentos em frases simples usando Inteligência Artificial.

As frases analisadas estão no arquivo `inputs/sentencas.txt`, e o resultado da análise foi obtido usando o recurso de **Análise de Sentimentos**.

---

## 📄 Sentenças analisadas

Algumas frases que utilizei no teste:

- Hoje estou me sentindo muito feliz!
- Tive um dia estressante no trabalho.
- Estou animada com meu futuro na área de RH!
- Nada deu certo hoje, que frustração!

---

## 🔍 Resultado da análise

A IA classificou as frases como **positivas**, **negativas** ou **neutras**, com base nas emoções transmitidas. A ferramenta também apresenta a **pontuação de confiança** para cada sentimento identificado.
